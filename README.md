## -Credit-Card-Transaction-Analysis
Analyzed credit card transactions to identify trends, customer behavior, and fraud indicators. Performed data transformation, aggregation, and visualization to deliver actionable insights through dashboards.
Credit Card Transaction Analysis

## **Project Overview:*
This project focuses on analyzing credit card transaction data to uncover spending patterns, customer behavior, and potential anomalies. The goal is to generate actionable insights that support better financial decision-making.

## **Objectives:*
Analyze customer spending behavior
Identify trends and patterns in transactions
Detect anomalies or unusual activities
Build interactive dashboards for insights

## **Tools & Technologies:*
Excel (Data Cleaning, Pivot Tables, Power Query)
Power BI (Dashboard & Visualization)
SQL (if used)

## **Key Analysis Performed:*
Data Cleaning & Transformation
Handling Missing Values & Errors
KPI Creation (Revenue, Transactions, Avg Spend)
Trend Analysis (Monthly / Region-wise)
Customer Segmentation
Fraud/Anomaly Detection

## **Dashboard Features:*
Transaction Overview
Spending Trends Over Time
Region-wise Performance
Customer Insights
